Editor für Hugo

2016-04-13 :: Philipp Weißmann
#Hugo 
Hugo ist eine schöne Software um Software um statische Webseiten und Blogs zu erstellen. Die in Go geschriebene Software ermöglicht das einfache Nutzen von Templates für neue Beiträge.
